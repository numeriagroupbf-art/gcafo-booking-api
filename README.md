# GCAFO Booking API

API REST pour l'application GCAFO Booking - Plateforme de réservation de services.

## 🚀 Fonctionnalités

- Authentification utilisateur
- Gestion des prestataires
- Système de réservation
- Avis et notations
- Messagerie en temps réel

## 📋 Prérequis

- Node.js 18+
- PostgreSQL
